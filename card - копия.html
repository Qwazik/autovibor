Pezzaioli SBA63 maialino

Полуприцеп Pezzaioli SBA63 maialino для перевозки исключительно свиней, поросят с 3-мя фиксированными поверхностями загрузки (этажами)

Цена: договорная

Технические характеристики

  • Длина 14,00м Ширина 2,55м Высота общая от земли 4,00м
  • Полуприцеп изготовлен из самонесущей конструкции полностью из алюминиевого антикоррозийного сплава.
  • Полы, шасси сделаны из антикоррозионного и противоскользящего алюминия
  • Пневматическая подвеска – с 3 неподвижными осями BPW с нагрузкой 9 т. Передняя ось подъемная с ручным управлением (поворотных осей нет)
  • 7 покрышек 385/65 R22.5 монтированные на стальные диски.
  • Опорный (стояночный) домкрат
  • Дисковая тормозная система c электронным управлением EBS Wabco-Westinghouse
  • Ручной саморегулируемый парковочный тормоз.
  • 3 фиксированных поверхности загрузки (этажа). Средняя часть этажей с открывающимися вручную вверх панелями.
  • Фиксируемая крыша. Центральная часть крыши открывается вручную
  • Крыша, изолированная с изолирующим материалом
  • Боковые стены из антикоррозийного алюминия
  • Внутренние заграждения: 5 перегородочные двери из алюминия на каждом этаже, что в конечном итоге делит каждый этаж на 6 отсеков(боксов)
  • Задняя загрузочная рампа (платформа): подъемная рампа – лифт оснащена с двух сторон 2 барьерами из алюминия. Грузоподъемность 1.500 кг.
  • 3 линии вентиляционных отверстий со сдвижными алюминиевыми панелями, открываемые/ закрываемые вручную
  • Гидравлическое оборудование: Независимая электрогидравлическая система (электро-гидронасос) для подъема рампы-лифта с аккумуляторами, емкостью для масла для гидравлики, и распределительный блок (без дистанционного управления).
  • Гидравлика оснащёна предохранительными клапанами, сертифицированными немецкой Ассоциацией Технического контроля (TUV), которые защищают платформу от непреднамеренного выхода из строя (падения платформ) в случае повреждения в системе гидравлики.
  • Вентиляция / Система принудительного проветривания: система состоит из вентиляторов (смонтированы на левой стороне) с функцией таймера для установки различных режимов на этажах.
  • Система водопоя: состоит их 3 линий-поилок с сосками-поилками на каждой стороне + душ — форсунки для свиней с баком воды и пневматикой
  • Система освещения: внутренняя система освещения + наружный задний фонарь.
  • Окраска (промышленная) в один цвет – красный RAL 3002 (или по желанию клиента)
  • 2 сливных трубы для сбросной воды с шаровым клапаном с тыльной стороны / крючки для прикрепления перевозимых транспортных средств / ящик для инструментов под шасси / задний бампер с системой освещения согласно правилам ЕЭС Устройство для подзарядки аккумуляторов на полуприцепе
